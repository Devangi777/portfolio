# Devangi Joshi - Portfolio Website

A modern, responsive portfolio website showcasing Devangi Joshi's skills, experience, and projects. Built with HTML, CSS, and JavaScript.

## 🚀 Features

- **Modern Design**: Clean, professional design with smooth animations
- **Responsive Layout**: Fully responsive design that works on all devices
- **Interactive Elements**: Hover effects, scroll animations, and smooth transitions
- **Mobile Navigation**: Hamburger menu for mobile devices
- **Smooth Scrolling**: Seamless navigation between sections
- **Skill Visualization**: Animated skill bars with progress indicators
- **Project Showcase**: Detailed project cards with technology tags
- **Contact Information**: Easy-to-find contact details with icons

## 🎨 Customization Guide

### Personal Information
Update the following sections in `index.html`:

1. **Hero Section** (lines 60-75):
   - Name and title
   - Personal description
   - Profile image (replace placeholder)

2. **About Section** (lines 85-110):
   - Personal bio
   - Academic achievements
   - Key statistics

3. **Education** (lines 120-160):
   - Degree information
   - Institution names
   - Dates and grades
   - Relevant coursework

4. **Skills** (lines 170-250):
   - Programming languages
   - Web technologies
   - Tools and frameworks
   - Skill proficiency levels

5. **Experience** (lines 260-290):
   - Job titles and companies
   - Duration and location
   - Responsibilities and achievements

6. **Projects** (lines 300-370):
   - Project names and descriptions
   - Technologies used
   - Key features and outcomes

7. **Contact Information** (lines 420-460):
   - Email, phone, location
   - Social media links

### Styling Customization

#### Colors
Update the color scheme in `styles.css`:

```css
/* Primary Colors */
--primary-color: #2563eb;
--secondary-color: #667eea;
--accent-color: #fbbf24;

/* Background Colors */
--bg-primary: #ffffff;
--bg-secondary: #f8fafc;
--bg-dark: #1f2937;
```

#### Fonts
Change the font family in `styles.css`:

```css
body {
    font-family: 'Your-Font-Name', sans-serif;
}
```

#### Animations
Customize animation speeds and effects in `script.js`:

```javascript
// Animation duration
const animationDuration = 2000; // milliseconds

// Scroll animation threshold
const observerOptions = {
    threshold: 0.1,
    rootMargin: '0px 0px -50px 0px'
};
```

## 🚀 Deployment

### GitHub Pages
1. Create a new repository on GitHub
2. Upload all files to the repository
3. Go to Settings > Pages
4. Select source branch (usually `main`)
5. Your portfolio will be available at `https://username.github.io/repository-name`

### Netlify
1. Drag and drop the portfolio folder to [Netlify](https://netlify.com)
2. Your site will be deployed instantly
3. Customize the URL in the site settings

### Vercel
1. Install Vercel CLI: `npm i -g vercel`
2. Run `vercel` in the portfolio directory
3. Follow the prompts to deploy
